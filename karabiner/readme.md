# Windows常用快捷键迁移

前置条件先安装brew，然后安装下面这个软件

```shell
brew install cask karabiner-elements
```

```shell
cd ~/.config/karabiner/assets/complex_modifications
```

然后中这个文件夹中添加本文件夹内的所有json并应用，其他设置见
